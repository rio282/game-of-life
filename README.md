# Game of Life

Conway's "[Game Of Life](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life)"
written in C using the new SDL version 3.

Keybinds:

| Key      | Action                  |
|----------|-------------------------|
| `ESCAPE` | Exit                    |
| `SPACE`  | Pause/Resume            |
| `R`      | Generate Random Pattern |
| `C`      | Clear Grid              |
| `G`      | Toggle Grid Drawing     |
